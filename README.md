# ESP32-Smart-Security-System
An IoT-based smart security system using ESP32 with motion detection, keypad authentication, LCD monitoring, and Blynk integration.

This is a simple IoT based security system made using ESP32.

The project was done to learn about:
- IoT
- Embedded systems
- Sensors
- Automation
- Blynk integration

## Features

- PIR motion detection
- Password unlock using keypad
- Buzzer alarm
- LCD display
- Manual ON/OFF button
- Mobile control using Blynk

## Components Used

- ESP32
- PIR Sensor
- 16x2 I2C LCD
- 4x4 Keypad
- Buzzer
- LED
- Push Button

## Working

When motion is detected, the buzzer turns ON and LCD shows alert message.

User can disable the alarm using:
- keypad password
- manual button
- Blynk app

## Applications

- Home security
- Smart monitoring
- Small office security
